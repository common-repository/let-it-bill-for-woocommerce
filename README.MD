== Installation ==

1. Navigate to the 'Add New' in the plugins dashboard
2. Search the LET IT BILL plugin
3. Click 'Install Now'
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= Is that plugin can work without WooCommerce? =

No, this plugin is a bridge between WooCommerce platform and LET IT BILL SaaS.
